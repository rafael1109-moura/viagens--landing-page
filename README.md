Projeto Agência de Viagens

Este projeto foi desenvolvido com o intuito de criar uma página web para uma agência de viagens, apresentando informações sobre um vilarejo europeu, contatos e dicas de viagem.

Objetivo

O objetivo principal é praticar a estruturação de páginas web utilizando HTML e CSS, aplicando conceitos de organização de conteúdo, navegação por âncoras e estilização básica.

Funcionalidades
Estrutura de página com seções informativas
Menu de navegação com links internos (âncoras)
Seções sobre experiência de viagem, contato e conselhos
Organização semântica do conteúdo
Tecnologias utilizadas
HTML5
CSS3
Estrutura do projeto
index.html — estrutura principal da página
css/style.css — estilização da página
